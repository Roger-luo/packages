rogerluo.dev

this page is made for convenience when someone (including myself) wants to check the status of packages I created/maintain.

Most of the packages are written in Julia Programming Language .

My Julia software projects are usually organized as many small packages so that people can easily use different components for other purpose.

The documentation of each project are maintained as a curated page under the project package. You can also find the link to them below.

QuantumBFS

yaoquantum.org
QuantumBFS is a github organization hosts the Yao project. The Yao project aims to build a framework for programming quantum devices in Julia.
  • Yao
    Extensible, Efficient Quantum Algorithm Design for Humans.
  • YaoArrayRegister
    Simulated Full Amplitude Quantum Register.
  • YaoBase
    Abstract type and interface definition for quantum circuit blocks.
  • YaoAPI
    APIs in Yao, minimum package for Yao compatibility.
  • YaoBlocks
    Standard basic quantum circuit simulator building blocks.
  • YaoSym
    Symbolic Computation Extension for Yao.
  • LuxurySparse
    A luxury sparse matrix package for Julia.
  • BitBasis
    Types and operations for basis represented by bits in linear algebra.
  • YaoLang
    The next eDSL for Yao and quantum programs in Julia.
  • YaoCompiler
    Unleash Julia on quantum devices.
  • BQCESubroutine
    Basic Quantum Circuit Emulation Subroutine.
  • YaoHIR
    High Level IR for YaoCompiler in native Julia.
  • YaoLocations
    Wire location types for quantum circuit.
  • YaoTargetQASM
    the QASM compilation target, includes QASM frontend and backend.
  • YaoTargetQobj
    the Qobj compilation target, includes IBM Q frontend and backend.
  • ZXCalculus
    An implementation of ZX calculus in native Julia.
  • OpenQASM
    The OpenQASM parser in native Julia.
  • IBMQClient
    The non-official IBM Quantum Experience REST API wrapper in native Julia.

Comonicon

comonicon.rogerluo.dev
Comonicon is a CLI (Command Line Interface) generator written in Julia and for Julia that aims to provide a fast, intuitive CLI generator for humans.
  • Comonicon
    The main package of this project.
  • ComoniconTypes
    Comonicon's Intermediate Representation for CLIs.
  • ComoniconTargetExpr
    Comonicon's code generator for native Julia Expr.
  • ComoniconOptions
    Options for Comonicon Applications.

Expronicon

expronicon.rogerluo.dev

  • Expronicon

  • ExproniconLite
